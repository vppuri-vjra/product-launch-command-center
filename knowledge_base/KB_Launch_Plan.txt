--- FEATURES IN THIS RELEASE ---

Feature 1: Redesigned Checkout Flow
- One-page checkout replaces 4-step funnel
- Apple Pay, Google Pay, PayPal added alongside credit card
- Address autofill via Google Places API
- Guest checkout no longer requires account creation
- A/B test target: 15% reduction in cart abandonment

Feature 2: Perks Loyalty Program
- Points per dollar: 1 point = $0.01 spend, redeemable at 500 points minimum
- Tiers: Bronze (0–499 pts), Silver (500–1999 pts), Gold (2000+ pts)
- Gold perks: free shipping, early access to sales, dedicated support line
- Real-time points balance in the app home screen
- Push notifications for point milestones

Feature 3: Push Notification Overhaul
- Granular opt-in by category (promotions, order updates, loyalty milestones)
- A/B test: personalized vs. generic copy — winner deployed at 50% rollout
